# 🛡️ 酒店民宿一物一码卫生溯源系统

三个独立页面，各司其职：

## 文件结构

| 文件 | 面向 | 用途 |
|------|------|------|
| `admin.html` | 管理员/老板 | 管理房间、用品、生成二维码、查看记录 |
| `staff.html` | 保洁人员 | 选房间→选用品→录入消毒/清洗记录 |
| `guest.html` | 住店客人 | 扫房间二维码，查看该房间所有用品卫生档案 |

## 使用流程

1. **管理员**打开 `admin.html` → 添加房间（支持批量）→ 自动生成用品 → 生成房间二维码 → 打印贴在房间内
2. **保洁员**打开 `staff.html` → 选择房间 → 选择具体用品 → 填写消毒/清洗记录 → 提交
3. **客人**扫描房间内二维码 → 自动打开 `guest.html` → 看到该房间所有杯子、床品、拖鞋的卫生记录

## 数据共享

三个页面共用浏览器 localStorage（key: `ht_db`），同一浏览器下数据互通。

## 部署

```bash
# 本地直接打开
open admin.html

# 或启动本地服务
python3 -m http.server 8080
# 管理端: http://localhost:8080/admin.html
# 保洁端: http://localhost:8080/staff.html
# 客人端: http://localhost:8080/guest.html?room=101
```

## 二维码指向

房间二维码扫码后跳转 `guest.html?room=房间号`，自动展示该房间卫生档案。
